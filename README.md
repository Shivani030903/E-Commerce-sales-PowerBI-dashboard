# E-Commerce-sales-PowerBI-dashboard

## Project Overview

This project analyzes Amazon E-Commerce sales data using Python and Power BI.

The objective is to identify sales trends, category performance, fulfillment efficiency, and customer order behavior through interactive visualizations.

---

## Tools Used

- Python
- Pandas
- NumPy
- Jupyter Notebook
- Power BI
- DAX

---

## Data Cleaning

Performed using Python:

- Missing value treatment
- Date formatting
- Data validation
- Data type conversions
- Duplicate checking

---

## Dashboard KPIs

- Total Revenue
- Total Orders
- Total Quantity
- Average Order Value
- Average Daily Revenue
- Cancellation Rate

---

## Dashboard Features

- Revenue by Month
- Revenue by Category
- Revenue by Fulfillment
- Revenue by State
- Orders by Status
- Orders by Service Level

## Key Insights

- T-Shirts generated the highest revenue.
- Maharashtra recorded the highest revenue among states.
- Amazon fulfillment generated the majority of sales revenue.
- Cancellation rate remained below 9%.
